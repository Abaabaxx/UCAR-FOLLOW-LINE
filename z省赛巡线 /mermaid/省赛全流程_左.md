```mermaid
stateDiagram-v2
    %% -- 状态定义 --
    FOLLOW_LEFT: 0-沿左墙循线
    STRAIGHT_TRANSITION: 1-直行过渡
    ROTATE_TO_FACE_EXIT_BOARD: 2-旋转正对出口板
    FOLLOW_LEFT_WITH_AVOIDANCE: 3-带避障循线
    AVOIDANCE_MANEUVER: 4-执行避障机动
    FOLLOW_TO_FINISH: 5-最终冲刺循线
    FINAL_STOP: 6-任务结束并停止

    %% -- 初始状态 --
    [*] --> FOLLOW_LEFT: 节点初始化

    %% -- 状态转换 --
    FOLLOW_LEFT --> STRAIGHT_TRANSITION: 检测到特殊区域\n(连续3帧边线在图像上部)
    STRAIGHT_TRANSITION --> ROTATE_TO_FACE_EXIT_BOARD: 边线足够近\n(Y坐标 > 阈值)
    ROTATE_TO_FACE_EXIT_BOARD --> FOLLOW_LEFT_WITH_AVOIDANCE: 已正对出口板
    FOLLOW_LEFT_WITH_AVOIDANCE --> AVOIDANCE_MANEUVER: 检测到障碍物
    AVOIDANCE_MANEUVER --> FOLLOW_TO_FINISH: 避障机动完成
    FOLLOW_TO_FINISH --> FINAL_STOP: 检测到停车区域
    FINAL_STOP --> [*]: 任务完成

    %% -- 状态行为说明 --
    note right of FOLLOW_LEFT
        <b>行为:</b>
        - <b>找到边线:</b> 执行PID单边线巡线
        - <b>丢失边线:</b> 停止 (发布速度0指令)
        <b>目标:</b> 沿左侧边线行驶，直到进入特殊区域
    end note

    note right of STRAIGHT_TRANSITION
        <b>行为:</b>
        - 保持直线前进
        - 同时持续进行视觉处理，寻找边线
        <b>目标:</b> 在进入环岛旋转前，先直行一段距离，确保车身姿态稳定
    end note

    note right of ROTATE_TO_FACE_EXIT_BOARD
        <b>行为:</b>
        - <b>未对齐时:</b> 以7度/秒的速度向左旋转
        - <b>已对齐时:</b> 转换到带避障循线状态
        <b>传感器:</b> 激光雷达 (LIDAR)
        <b>目标:</b> 旋转车身，直到正对出口板
        <b>检测参数:</b>
        - 扫描范围: 正前方±50度
        - 板子长度: 0.4~0.6米
        - 角度容忍: ±9度(垂直)
    end note

    note left of FOLLOW_LEFT
        <b>全局控制:</b>
        - 可随时通过ROS Service /follow_line/run 启停
        - 视觉系统检测特殊区域以触发第一次状态转换
        - 激光雷达用于在第二、三、四、五阶段进行精确对齐和位置控制、障碍物检测和自主避障
    end note

    note right of FOLLOW_LEFT_WITH_AVOIDANCE
        <b>行为:</b>
        - 与`FOLLOW_LEFT`状态相同，执行PID单边线巡线
        - 额外使用Lidar持续检测前方障碍物
        <b>传感器:</b> 视觉 (循线) + Lidar (避障)
        <b>目标:</b> 安全地沿左墙行驶，直到遇到障碍物
        <b>避障参数:</b>
        - 监控范围: 前方±20度
        - 触发距离: < 0.4米
    end note

    note right of AVOIDANCE_MANEUVER
        <b>行为 (分三步):</b>
        - 1. 向左平移 0.5米
        - 2. 向前直行 0.58米
        - 3. 向右平移 0.5米
        <b>传感器:</b> 里程计 (Odometry)
        <b>目标:</b> 完全基于里程计数据，绕开障碍物
        <b>特点:</b> 此状态不依赖视觉或Lidar进行导航
    end note

    note right of FOLLOW_TO_FINISH
        <b>行为:</b>
        - <b>循线:</b> 使用双边线检测算法，沿赛道中心线行驶
        - <b>检测:</b> 同时检测图像底部的停车区域
        <b>传感器:</b> 视觉
        <b>目标:</b> 快速稳定地冲向终点，直到检测到停车标志
        <b>停车检测:</b>
        - 窗口: 图像底部 3x30 像素
        - 阈值: 白色像素占比 > 60% (连续3帧)
    end note

    note right of FINAL_STOP
        <b>行为:</b>
        - 发布速度(0,0,0)指令
        - 设置 is_running=False 停止主循环
        <b>目标:</b> 机器人完全停止，任务结束
    end note
```
